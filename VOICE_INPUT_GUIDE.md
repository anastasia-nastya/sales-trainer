# 🎤 РУКОВОДСТВО ПО ГОЛОСОВОМУ ВВОДУ

## Быстрая инструкция

### 1. Откройте в правильном браузере

✅ **Google Chrome** (Windows/Mac/Android) — ЛУЧШИЙ ВЫБОР
✅ **Safari** (Mac/iPhone)
✅ **Microsoft Edge**

❌ Firefox (может не работать)
❌ Opera (может не работать)

### 2. Разрешите микрофон

При первом нажатии на кнопку 🎤 появится запрос:

```
┌─────────────────────────────────┐
│ "localhost:3000" хочет          │
│ использовать микрофон           │
│                                 │
│  [БЛОКИРОВАТЬ]    [ВЫПОЛНИТЬ]   │
└─────────────────────────────────┘
```

**ОБЯЗАТЕЛЬНО нажмите "ВЫПОЛНИТЬ"!**

### 3. Если заблокировали ранее

#### В Chrome/Edge:

**Способ 1 — Быстрый:**
1. Посмотрите на адресную строку (там где написано localhost:3000)
2. Найдите значок 🔒 (замок) или ⚠️
3. Нажмите на него
4. Найдите "Микрофон" → нажмите "Разрешить"
5. Перезагрузите страницу (F5)

**Способ 2 — Через настройки:**
1. Нажмите три точки ⋮ в правом верхнем углу
2. "Настройки" → "Конфиденциальность и безопасность"
3. "Настройки сайтов" → "Дополнительные настройки сайтов"
4. "Микрофон" → найдите "localhost:3000" → "Разрешить"
5. Перезагрузите страницу

#### В Safari:

1. Адресная строка → нажмите на 🔒 или ⓘ
2. Найдите "Микрофон" → "Разрешить"
3. Перезагрузите страницу

## Как пользоваться голосовым вводом

### Запись голоса:

1. Нажмите на **кнопку микрофона** 🎤
2. Подождите 1 секунду (появится "Запись...")
3. **Говорите чётко** (на русском языке)
4. Сделайте паузу на 1-2 секунды
5. Текст появится автоматически и отправится

### Пример фраз для тестирования:

- "Здравствуйте!"
- "Меня зовут Александр"
- "Рад встрече"
- "Давайте обсудим вашу компанию"

## Советы для лучшего распознавания

✅ Говорите **чётко и размеренно**
✅ Используйте **хороший микрофон**
✅ Минимизируйте **фоновый шум**
✅ Находитесь **близко к микрофону**
✅ Делайте **паузы** между фразами
✅ Используйте **простые предложения**

## Если голос не работает

### Проверка 1: Микрофон работает?

1. Откройте настройки системы
2. Проверьте уровень микрофона
3. Попробуйте записать голосовую заметку

### Проверка 2: Браузер поддерживается?

Откройте консоль браузера (F12) и введите:

```javascript
'speechRecognition' in window || 'webkitSpeechRecognition' in window
```

Если `true` — поддерживается, если `false` — смените браузер на Chrome.

### Проверка 3: Разрешения даны?

F12 → Console → введите:

```javascript
navigator.permissions.query({name: 'microphone'})
  .then(p => console.log('Microphone:', p.state))
```

Должно быть: `"granted"` или `"prompt"`

## Troubleshooting

### "Доступ к микрофону запрещён"

✅ Разрешите в настройках браузера (см. выше)
✅ Перезагрузите страницу (F5)

### "Распознавание речи не поддерживается"

✅ Откройте в Google Chrome
✅ Обновите браузер до последней версии

### "Речь не обнаружена"

✅ Говорите громче и чётче
✅ Проверьте, что микрофон работает
✅ Уберите фоновый шум

### Текст не появляется

✅ Дождитесь паузы 1-2 секунды
✅ Попробуйте сказать короче фразу
✅ Нажмите микрофон снова

## Альтернатива: Диктофон

Если голосовой ввод не работает:

1. Используйте диктофон на телефоне
2. Запишите голос → конвертируйте в текст
3. Вставьте текст в поле ввода
