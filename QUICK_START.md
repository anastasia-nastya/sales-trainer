## Следующие шаги

### 1️⃣ Добавьте API ключ Anthropic

Создайте файл `.env.local` в корне проекта:

```bash
cd /Users/anastasia/projects/sales-trainer
nano .env.local
```

Добавьте ваш API ключ:

```env
ANTHROPIC_API_KEY=sk-ant-your-actual-api-key-here
```

**Как получить API ключ:**
1. Перейдите на https://console.anthropic.com/
2. Войдите или создайте аккаунт
3. Нажмите "Get API Key"
4. Скопируйте ключ

### 2️⃣ Проверьте приложение

Dev-сервер уже запущен на http://localhost:3000

Откройте браузер и перейдите по адресу.

### 3️⃣ Попробуйте тренажёр

1. **Главная страница** — выберите сценарий
2. **Страница сценария** — изучите детали
3. **Тренировка** — начните диалог с AI-клиентом

## 🎯 Что уже работает

### ✅ Функциональность

- **Главная страница** с 6 сценариями (2 встречи, 2 возражения, 2 переговоры)
- **Страницы сценариев** с деталями и персоной клиента
- **Аудио диалоги** — распознавание речи (Web Speech API)
- **AI-клиент** — генерация ответов через Claude API
- **Оценка диалогов** — анализ и рекомендации
- **Результаты тренировки** — детальный отчёт
- **Дашборд** — статистика и прогресс
- **История** — все тренировки
- **Профиль** — достижения и настройки

### ⚠️ Требуется

- **API ключ Anthropic** для работы AI (без него диалоги не будут работать)
- **Разрешение микрофона** в браузере для голосового ввода

## 🆘 Возможные проблемы

### 1. API ключ не работает
**Решение:** Проверьте, что ключ правильный в `.env.local` и файл существует

### 2. Микрофон не работает
**Решение:** Проверьте разрешения браузера на доступ к микрофону

### 3. Распознавание речи не работает
**Решение:** Web Speech API поддерживается не во всех браузерах. Попробуйте Chrome.

### 4. Ошибка компиляции
**Решение:** Проверьте вывод dev-сервера на наличие ошибок

## 📞 Поддержка

При возникновении проблем проверьте:
1. Dev-сервер запущен (`npm run dev`)
2. API ключ добавлен в `.env.local`
3. Правильный браузер (Chrome/Safari/Edge)
4. Консоль браузера на ошибки
